El programa se puede ejecutar siguiendo los siguientes pasos:
1. Cree si no existe o rellene el archivo .properties.txt con las rutas donde se guardaran los archivos de persistencia de datos. Ejemplo:
```
pathActivityRep=persistence/activity.txt
pathAssistantRep=persistence/assistant.txt
pathCampRep=persistence/camp.txt
pathInscriptionRep=persistence/inscription.txt
pathMonitorRep=persistence/monitor.txt
```

2. Una vez hecho esto ejecute:
```
java -jar cli-summer-camp-management.jar
```
